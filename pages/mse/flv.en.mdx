## Using FLV plugin

**If you want to play flv video, you need to embed flv.js first.**

**[flv.js](https://github.com/bilibili/flv.js)**

DEMO
```html
<script src="https://cdn.jsdelivr.net/npm/flv.js@1.6.2/dist/flv.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/@wenzi7777/mplayer@latest/dist/mplayer.bundle.min.js"></script>
<div id={'mplayer'}></div>
<script>
    let options = {
        videos: [
            {
                title: 'video title',
                src: 'video src.flv'
            }
        ]
    }
    const mplayer = new MPlayer(document.querySelector('#mplayer'), options)
</script>
```

flv videos **do not support** multi-resolution.
